## Arduino kullanılarak yapılmış bir termostat örneği
<br>
<img src="https://github.com/yedincel/arduino-thermostat/blob/main/simulation.png">

Gerekli malzemeler:
- Arduino Uno 
- DHT22 Nem ve Sıcaklık Sensörü
- LCD Ekran
- 3 Adet LED: Kırmızı, mavi ve sarı
- Dirençler
- Jumper Kablolar
